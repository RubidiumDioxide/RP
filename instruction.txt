Для запуска достаточно собрать и запустить проект Asp net core, я запускала из Visual Studio. 

В корневой папке проекта лежит input.txt с данными, на которых тестировался api. 

По умолчанию при запуске проекта откроется swagger, в котором можно протестировать оба get-метода контроллера напрямую. 

Если перейти на https://localhost:7125/app, откроется очень условный веб-клиент, где с помощью fetch-ей посылаются запросы серверу, получается и отображается ответ. 

Данные о рекламных площадках хранятся в виде дерева, структура и ее методы описаны в папке Structure. 

Для хранения дерева, образованного загруженными из файла данными, используется статический класс Data с единственным праметром типа Tree.
